# COVID-19 Data Analysis

## Introduction

COVID-19 has affected people all around the world. In this project, I have analyzed COVID-19 data to understand how the virus spread across different countries and how the number of active, recovered, and death cases changed.

## About the Project

This project is mainly focused on exploring and visualizing COVID-19 data using Python. The data is cleaned and processed, and then different visualizations are created to better understand the situation.

## Tools and Technologies

* Python
* Pandas
* Plotly
* Google Colab

## What I Did

* Collected and combined COVID-19 dataset
* Cleaned and prepared the data using Pandas
* Created interactive charts using seaborn
* Built a world map to show active cases across countries

## Results

The visualizations clearly show how COVID-19 cases are distributed across the world. Some countries have higher active cases compared to others, and this can be easily understood through the maps and charts.

## Conclusion

This project helped me understand how data analysis works in real-world situations. It also showed how visualization tools can make data easier to understand and more meaningful.
